# go-exec

This repository contains solutions to exercises focused on **Go internals**.  
You can find examples of channel usage, goroutines, concurrency patterns, and more.

---

## List of Exercises
| Status | Name | Difficulty | Description |
|--------|------|------------| ----------- |
| ✅ | Fan-In Merge         | Medium | Redirect information from several channels to the result channel. |
| ✅ | Slice Intersection   | Easy   | Find intersection between 2 slices and return new slice as a result. |
| ❌ | Noisy Data Pipeline  | Hard   | Transfer data and validate it using channels and context. |
| ❌ | Timeout Wrapper      | Easy   | Create wrapper function that interrupts execution of the main one after some time. |
